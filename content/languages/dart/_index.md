Dart can print a string:

~~~dart
main() {
   print('Sunday'); // Sunday
}
~~~

but it seems to have no way to print a string representation. Other languages
can do this, like Python:

~~~py
s1 = repr('Sunday')
print(s1) # 'Sunday'
~~~
