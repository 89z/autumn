Letter | Type
-------|-----
`a`    | Array
`b`    | Boolean
`e`    | Error
`m`    | Map
`n`    | Number
`o`    | Object
`r`    | Resource
`s`    | String
`t`    | Set (`t` in `set`, also `t` in `true`)
`y`    | Byte (`y` in `byte`)
