D cURL static

Wed Mar 4 18:45:59 2020

The first choice for this task is std.net.curl. However D does not provide a static library:

https://issues.dlang.org/show_bug.cgi?id=15841

We can build cURL ourself:

./buildconf
./configure --host=x86_64-w64-mingw32 --disable-shared
make LDFLAGS=-all-static
