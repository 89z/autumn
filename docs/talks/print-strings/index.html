Print strings

Wed Mar 4 18:45:59 2020

For output, PHP has echo and print, They require explicit newline:

<?php
$s1 = 'sunday';
# example 1
print $s1 . "\n";
# example 2
echo $s1 . "\n";
# example 3
echo $s1, "\n";
# example 4
print "$s1\n";
# example 5
echo "$s1\n";

and Go does not support interpolation:

package main
import "fmt"
func main() {
   s1 := "sunday"
   fmt.Print(s1, "\n")
   fmt.Print(s1 + "\n")
   fmt.Printf("%v\n", s1)
}

To save time, with each language we should just use the shortest method available.
