HTTP authorization header

Mon Apr 27 18:46:46 2020 network

Go

https://golang.org/pkg/net/http#Request.SetBasicAuth

PHP

<?php
$s1 = 'https://api.github.com/rate_limit';
# example 1
$s2 = getenv('USER');
$s3 = getenv('PASS');
$r1 = curl_init($s1);
curl_setopt($r1, CURLOPT_USERAGENT, 'anonymous');
curl_setopt($r1, CURLOPT_USERPWD, $s2 . ':' . $s3);
curl_exec($r1);
# example 2
$r2 = curl_init($s1);
curl_setopt($r2, CURLOPT_USERAGENT, 'anonymous');
curl_setopt($r2, CURLOPT_NETRC, true);
curl_exec($r2);

Some examples require php-curl package.

https://php.net/function.curl-setopt
