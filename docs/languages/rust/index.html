Rust

Rust does not have Map literals:

https://github.com/rust-lang/rfcs/issues/542

but it does have struct:

fn main() {
   #[derive(Debug)]
   struct Pref {
      k: String,
      n: u8,
   }
   let o1 = Pref { k: "Sun".to_string(), n: 10 };
   println!("{:?}", o1);
}

https://doc.rust-lang.org/book/ch05-02-example-structs.html

and match:

https://doc.rust-lang.org/book/match.html

Go struct allows missing fields:

package main
import "fmt"
func main() {
   type Pref struct {
      k string
      n int
      b bool
   }
   o1 := Pref { k: "Sun", n: 10 }
   fmt.Printf("%+v\n", o1)
}

However Rust does not:

fn main() {
   #[derive(Debug)]
   struct Pref {
      k: String,
      n: u8,
      b: bool
   }
   // error[E0063]: missing field `b` in initializer of `main::Pref`
   let o1 = Pref { k: "Sun".to_string(), n: 10 };
   println!("{:?}", o1);
}

Instead requiring something like this:

fn main() {
   #[derive(Debug, Default)]
   struct Pref {
      k: String,
      n: u8,
      b: bool
   }
   let o1 = Pref { k: "Sun".to_string(), n: 10, ..Default::default() };
   println!("{:?}", o1);
}
