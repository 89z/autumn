Compiler emits no warnings or errors

fn main() {
   mut m2 := {'Sun': 10, 'Mon': 11}
   m2['Tue'] = 12
   println("Sunday")
}

If I remove mut, it will cause a problem:

fn main() {
   m2 := {'Sun': 10, 'Mon': 11}
   m2['Tue'] = 12
   println("Sunday")
}

but instead of printing a warning or error, the compiler just exits. Does V have a way to emit messages when a problem happens? Workaround is Self-compiled binary.
