Iterate a file

Wed Mar 4 18:45:59 2020 iterate input-output

D

https://dlang.org/phobos/std_stdio#.lines

PHP

<?php
$r1 = fopen('a.txt', 'r');
while (true) {
   $s1 = fgets($r1);
   if (feof($r1)) {
      break;
   }
   var_dump($s1);
}

feof is triggered after fgets. Further, trim expects a string, so we cannot use it until we are certain that we have a string.

https://php.net/function.feof
