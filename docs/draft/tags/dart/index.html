Dart language

Sun May 24 2020

Other languages like Python can print a string representation:

s = repr('Sunday')
# 'Sunday'
print(s)

Dart cannot do this, but you can use boolean as workaround:

main() {
   var s = 'Sunday';
   // true
   print(s == 'Sunday');
}
