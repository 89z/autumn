Array Map Set

Sun Apr 12 12:22:05 2020

For Array, use string values. For Maps, use string keys and number values. For Sets, use string keys and boolean values.

package main
import "fmt"
func main() {
   a1 := []string{"Sun", "Mon"}
   m1 := map[string]int{"Sun": 10, "Mon": 11}
   c2 := map[string]bool{"Sun": true, "Mon": true}
}

We can remember this, as each value starts with a string.
