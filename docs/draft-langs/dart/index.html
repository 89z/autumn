Dart

Sun Apr 26 20:40:55 2020

Dart can print a string:

main() {
   print('Sunday'); // Sunday
}

but it seems to have no way to print a string representation. Other languages can do this, like Python:

s1 = repr('Sunday')
print(s1) # 'Sunday'
