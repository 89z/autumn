Talk / Exit program

Sun Oct 11 2020

Rust needs a single line way to exit a program. With Go, you have Log.Fatal:

package main

import (
   "log"
   "os"
)

func main() {
   o, e := os.Stat("a.go")
   if e != nil {
      log.Fatal(e)
   }
   n := o.Size()
   println(n)
}

This works for a Result:

use std::{
   fs,
   io
};

fn main() -> io::Result<()> {
   let o = fs::metadata("a.rs")?;
   let n = o.len();
   println!("{}", n);
   Ok(())
}

This works for Option:

use std::char;

fn main() -> Result<(), u32> {
   let n = 10;
   let c = char::from_digit(n, 16).ok_or(n)?;
   println!("{}", c);
   Ok(())
}

but what if you have both in the same function? Normally we could use this:

use std::error::Error;

fn main() -> Result<(), Box<dyn Error>> {
   Err("March")?
}

but it doesnt work for numbers:

use std::error::Error;

fn main() -> Result<(), Box<dyn Error>> {
   Err(9)?
}
