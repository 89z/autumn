Byte

If we use a1 for Byte Arrays, it will be confusing when used next to traditional arrays:

package main
import "fmt"
func main() {
   // example 1
   a1 := []string{"Sun", "Mon"}
   fmt.Println(a1)
   // example 1
   a2 := []byte{'S', 'u', 'n'}
   fmt.Printf("%s\n", a2)
}

If we use s1 for Byte Array, it will be confusing when used next to traditions strings:

package main
import "fmt"
func main() {
   // example 1
   s1 := "Sunday"
   println(s1)
   // example 2
   s2 := []byte{'S', 'u', 'n'}
   fmt.Printf("%s\n", s2)
}

https://golang.org/ref/spec#Numeric_types
