Fri Sep 18 2020

If we do our base with UTC, it will be the same short date as my time zone:

<?php
$n = 1577836799;
# example 1
$s1 = date(DATE_W3C, $n);
# example 2
date_default_timezone_set('America/Chicago');
$s2 = date(DATE_W3C, $n);
# print
var_dump(
   $s1 == '2019-12-31T23:59:59+00:00',
   $s2 == '2019-12-31T17:59:59-06:00'
);

If we do our base with my time zone, short dates will be different. This is good, as I will know what zone functions are returning:

<?php
$n = 1577858399;
# example 1
$s1 = date(DATE_W3C, $n);
# example 2
date_default_timezone_set('America/Chicago');
$s2 = date(DATE_W3C, $n);
# print
var_dump(
   $s1 == '2020-01-01T05:59:59+00:00',
   $s2 == '2019-12-31T23:59:59-06:00'
);
