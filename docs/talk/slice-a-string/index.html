Talk:Slice a string

Tue May 5 2020

We want to return begin character(s), as syntax can be different:

package main
func main() {
   s1 := "Sunday"
   s2 := s1[:1]
   println(s2)
}

We want to return middle character(s), as Go can have unexpected results:

package main
import "fmt"
func main() {
   s1 := "Sunday"
   s2 := s1[1:1]
   fmt.Printf("%q\n", s2) // ""
}

We also want to return end character(s), as Go can have unexpected results:

package main
func main() {
   s1 := "Sunday"
   s2 := s1[-1:]
   println(s2) // invalid slice index -1 (index must be non-negative)
}

We also want to return a single character, as Go can have unexpected results:

package main
func main() {
   s1 := "Sunday"
   s2 := s1[0]
   println(s2) // 83
}

We also want to return multiple characters, as JavaScript can have unexpected results:

let s1 = 'Sunday';
let s2 = s1[:2]; // SyntaxError: Unexpected token ':'
console.log(s2);

Make the begin a single character:

let s1 = 'Sunday';
let s2 = s1.charAt();
console.log(s2);
