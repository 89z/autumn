The current PHP options are garbage for Windows. Example 1:

<?php
$s1 = 'https://example.com/sun"mon';
$s2 = escapeshellarg($s1);
var_dump($s2 == '"https://example.com/sun mon"');

Example 2:

<?php
$s1 = 'https://example.com/sun"mon';
$s2 = escapeshellcmd($s1);
var_dump($s2 == 'https://example.com/sun^"mon');

The second example seems promising, but if you try this:

<?php
$s1 = 'https://example.com/sun"mon';
$s2 = escapeshellcmd($s1);
system('firefox ' . $s2);

The quote is stripped and you get here:

https://example.com/sunmon

The workaround is to split on double quotes, then join again with double quotes. Interestingly, interleaving quotes do not need to be escaped:

https://stackoverflow.com/questions/7760545
