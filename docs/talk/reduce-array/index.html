Talk / Talk:Reduce array

Tue Oct 6 2020

We can approach this three ways. Using the reduce function:

<?php
$a = ['May', 'June'];
$f = fn ($sa, $sc) => $sa . $sc;
$s = array_reduce($a, $f);
var_dump($s == 'MayJune');

Functional loop:

package main

func f(sa, sc string) string {
   return sa + sc
}

func main() {
   a := []string{"May", "June"}
   s := ""
   for n := range a {
      sc := a[n]
      s = f(s, sc)
   }
   println(s == "MayJune")
}

<?php
$a = ['May', 'June'];
$f = fn ($sa, $sc) => $sa . $sc;
$s = '';

foreach ($a as $sc) {
   $s = $f($s, $sc);
}

var_dump($s == 'MayJune');

Inline loop:

package main

func main() {
   a := []string{"May", "June"}
   s := ""
   for n := range a {
      s += a[n]
   }
   println(s == "MayJune")
}

<?php
$a = ['May', 'June'];
$s = '';

foreach ($a as $sc) {
   $s .= $sc;
}

var_dump($s == 'MayJune');

The reduce function can be used in cases where you dont need or dont care about the current index, and just want some short code.

The inline loop can be used in cases where reduce function is not available (Go), or in cases wheres reduce callback does not offer current index (PHP).

The functional loop can be used when you need a loop, but you want to reuse the inner code.
