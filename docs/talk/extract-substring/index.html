Talk / Talk:Extract substring

Tue Oct 6 2020

Points of ambiguity:

  1. How many characters are returned?
  2. Is second argument length, inclusive or exclusive?

First question is solved by showing the result, example with JavaScript:

let s = 'May';
let s1 = s.slice(1);
console.log(s1 == 'ay');

For second question, we could try some examples. This fails as it could be length or exclusive:

let s = 'June';
let s2 = s.slice(0, 1);
console.log(s2 == 'J');

This fails as it could be length of exclusive:

let s = 'June';
let s2 = s.slice(0, 2);
console.log(s2 == 'Ju');

This is good:

let s = 'June';
let s2 = s.slice(1, 2);
console.log(s2 == 'u');

This is good:

let s = 'June';
let s2 = s.slice(1, 3);
console.log(s2 == 'un');
