Object contains property

Fri Jun 19 2020 object

JavaScript

class Day {
   Sun = 10;
   Mon = 11;
}
let o1 = new Day;
// example 1
let b1 = 'Sun' in o1;
// example 2
let b2 = o1.hasOwnProperty('Sun');
// print
console.log(b1, b2);
  • https://developer.mozilla.org/Web/JavaScript/Reference/Global_Objects/Object/hasOwnProperty
  • https://developer.mozilla.org/Web/JavaScript/Reference/Operators/in

PHP

<?php
$o = new class {
   public $Sunday = 10;
};
# example 1
$b1 = property_exists($o, 'Sunday');
# example 2
$b2 = array_key_exists('Sunday', $o);
# print
var_dump($b1, $b2);

For backward compatibility reasons, array_key_exists will also return TRUE if key is a property defined within an object given as array.

  • https://php.net/function.property-exists
